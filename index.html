Digital Scale | Mini Project

Digital Scale is a measuring tool that utilizes ULTRA Sonic Sensor to measure length and display it on an LED screen. It works by sending a sound pulse toward the target and then measuring the time it takes for the sound echo to return and calculates the distance. This tool can be used at any time, even in dark places, since it works on an ultrasonic sensor and requires no light.

Nail Deformities Detection | Main Project

This project focuses on detecting nail deformities through image processing using various algorithms such as CNN, KNN, Decision Tree, and K-Nearest. Among these, CNN demonstrates the highest accuracy in detecting nail deformities.
